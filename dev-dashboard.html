  6. Test locally using the button above
  7. Deploy to GitHub Pages when ready

💡 Development Tips

  • Local Testing: Run a local server (required for camera access)
  • HTTPS Required: Camera only works with HTTPS (GitHub Pages provides this)
  • Mobile First: Test primarily on smartphones
  • Good Lighting: Ensure your map is well-lit for tracking
  • Image Quality: Use high-contrast, sharp images for best results

💻 Local Server Commands

To test locally, run one of these commands in your terminal:

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (if you have it)
npx http-server -p 8000

# PHP (if you have it)
php -S localhost:8000

Then visit: http://localhost:8000

Note: Camera access might not work on localhost. For full testing, deploy to GitHub Pages.
